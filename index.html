Wine quality API Documentation

Welcome to the API documentation for our wine quality prediction AP-AI.

What is AP-AI? It's a new beautiful AI model wrap into an API to help you to build the best products.

Endpoints

POST /predict

This endpoint allows you to make a prediction of the wine quality according to chemical composition of a given wine.

How to request this endpoint?

You should request this endpoint using a POST method with a JSON content.

Your JSON should contain at least : "input" key with associated list of [1 - n] lists of 11 float values, such as : 

                
    {
      "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]
    }

In which :


7.0 is the fixed acidity value
0.27 is the volatile acidity value
0.36 is the citric acid concentration
20.7 is the residual sugar value
0.045 is the concentration in chlorides
45.0 is the free sulfur dioxide value
170.0 is the total sulfur dioxide value
1.001 is the density
3.0 is the pH
0.45 is the sulphates concentration
8.8 is the alcohol concentration

Example, with unique input (such as above) : 

            
    $ curl -i -H "Content-Type: application/json" -X POST -d '{
                                                                "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]
                                                              }' https://winequality-api-esequeiros.herokuapp.com/predict

It returns a JSON like so: 

            
    {'prediction': [6.0]}

Example, with several inputs : 

              
      $ curl -i -H "Content-Type: application/json" -X POST -d '{
                                                                  "input": [
                                                                  [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]
                                                                  [5.0, 0.98, 0.32, 18.9, 0.050, 75.0, 122.0, 0.401, 3.1, 0.21, 1.2]
                                                                  ]
                                                                }' https://winequality-api-esequeiros.herokuapp.com/predict

It returns a JSON like so: 
            
    {'prediction': [6.0, 5.0]}

Errors

Our API will return an error with format: 

            
    {
      "msg": a short description
    }

Identified errors:

  • Missing JSON
